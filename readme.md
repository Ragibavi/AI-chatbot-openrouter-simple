## ⚙️ Cara Instalasi dan Menjalankan

1. **Clone repository ini:**

```bash
git clone https://github.com/Ragibavi/AI-chatbot-openrouter-simple.git
cd AI-chatbot-openrouter-simple
```

2. **Instalasi dependensi:**

```bash
npm install
```

3. **Jalankan aplikasi:**

```bash
node app.js
```

4. **Buka di browser:**

```
http://localhost:3000
```
